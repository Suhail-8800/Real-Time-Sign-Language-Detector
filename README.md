# Sign Language Detection Project

This is a real-time Sign Language Detection system using
MediaPipe and an LSTM model.

## Features
- Real-time webcam-based sign detection
- LSTM-based sequence prediction
- Sentence refinement

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Download the trained model (not included in repo)

3. Run:
   python live_demo.py
